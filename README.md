### 📍 [Paseo · iOS Travel Photo Timeline  ](https://github.com/bowenbzhen/paseo) 

An iOS app that transforms your travel photos into a chronological visual timeline, automatically organized by **location** and **date**.  

Instead of scrolling through thousands of camera roll photos, Paseo curates your journey into a clean travel log:  
- where you went  
- when you were there  
- what you saw  

Built for travelers who want to **relive and share trips** without the manual effort of journaling or scrapbooking.

### 🔍 [Visual Knowledge Explorer](https://github.com/bowenbzhen/visual-knowledge-explorer) 

A visual knowledge expansion tool: type in a concept, and it generates a composed poster showcasing its **variants or subtypes**, each paired with a clean illustration and label.  

Think of it as a visual taxonomy generator:
- Type *"designer chairs"* → get a poster with Eames, Wishbone, Barcelona, and more.  
- Type *"chili peppers"* or *"dog breeds"* → instantly see the diversity of a category laid out visually.  

It’s part reference poster, part learning tool, making it easier to **explore, compare, and remember** the structure of almost anything.


### ✏️ [Excalidraw Notes Skills · Diagrams that live next to your markdown](https://github.com/bowenbzhen/excalidraw-notes-skills)

A small CLI (`mdex`) and matching agent skill that make Excalidraw diagrams behave like first-class citizens in markdown.

The workflow is intentionally boring:
- keep a `.excalidraw` source file beside your markdown
- build it to `.svg` with `mdex`
- link the SVG from markdown like any other image

The result renders cleanly in **GitHub, Obsidian, VS Code preview, and static site generators**, while the source stays editable at [excalidraw.com](https://excalidraw.com). Ships as a skill for **Claude Code** and **OpenAI Codex**, plus a Python builder template for dense study-note scenes from videos, papers, and articles.
